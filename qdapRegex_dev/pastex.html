Paste Regular Expressions

Usage

pastex(..., dictionary = getOption("regex.library"))

Arguments

...
Ray regular expressions to paste together or a named expression from the default regular expression dictionary prefixed with single at (@) (e.g., "@rm_hash") or a regular expression from regex_supplement dictionary prefixed with double at (@@) (e.g., "@@time_12_hours").
dictionary
A dictionary of canned regular expressions to search within.

Paste Regular Expressions

Value

Returns a single string of regular expressions pasted together with pipe(s) |

Description

A wrapper for paste(collapse="|") that also searches the default and supplemental (regex_supplement) dictionaries for regular expressions before pasting them together with a pipe | separator.

Examples

x <- c("There is $5.50 for me.", "that's 45.6% of the pizza",
    "14% is $26 or $25.99", "It's 12:30 pm to 4:00 am")

pastex("@rm_percent", "@rm_dollar")



[1] "\\(?[0-9.]+\\)?%|\\$\\(?[0-9.]+\\)?"



pastex("@rm_percent", "@@time_12_hours")



[1] "\\(?[0-9.]+\\)?%|(1[012]|[1-9]):[0-5][0-9](\\s?)(am|pm)"




rm_dollar(x, extract=TRUE, pattern=pastex("@rm_percent", "@rm_dollar"))
