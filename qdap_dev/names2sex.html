Names to Gender Prediction

Usage

names2sex(names.list, pred.sex = TRUE)

Arguments

names.list
Character vector containing first names.
pred.sex
logical. If TRUE overlapping M/F names will be predicted based on highest cummulative frequency.

Value

Returns a vector of predicted gender (M/F) based on first name.

Description

Predict gender from U.S. names (based on 1990 U.S. census data).
