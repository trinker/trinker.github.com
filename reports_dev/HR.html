Convert path/url to HTML href Tag

Usage

HR(path = "clipboard", text = NULL, copy2clip = TRUE, print = FALSE)

Arguments

text
character vector of text to hyperref from. Defualt uses the basename of the path.
path
character vector url/path copied to the clipboard. Default is to read from the clipboard. Note that Windows users do not have to reorient slashes in local paths if reading from the clipboard.
copy2clip
logical. If TRUE attempts to copy the output to the clipboard.
print
logical. If TRUE cat prints the output to the console. If FALSE returns to the console.

Value

Returns a character vector of an HTML href tag.

Description

Wrap a path/url to generate an HTML href tag.

Examples

## HR("assets/img/fry_admin_1.mp4","new")

HR(path="http://dl.dropbox.com/u/61803503/Likert.pdf", print = TRUE)
