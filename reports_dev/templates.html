View Templates

Usage

templates(print.cons = TRUE, names = TRUE)

slidify_templates()

Arguments

print.cons
logical. If TRUE The template information id printed to the console.
names
logical. If TRUE only the names of the templates will be returned.

Description

templates - View all the reports package templates.

slidify_templates - View all the built in slidify templates.

Details

templates - prints to the console to view possible templates (feel free to submit your own; see doc_temp for details) as well as optionally returning a vector of names that the user can pass to other functions.

slidify_templates - prints a list of minimal and full .Rmd files included with the reports package for use in presentations created by the slidify package. These can be passed to the slidify argument of new_reports and presentation.

Examples

## Not run: # templates()# templates(FALSE)# new_report("new", templates(FALSE)[4])# # slidify_templates()# ## End(Not run)
