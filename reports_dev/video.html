Video (YouTube/Vimeo) url to HTML iframe Tag

Usage

YT(path = "clipboard", copy2clip = TRUE, width = 640, height = 360, print = FALSE)

VM(path = "clipboard", copy2clip = TRUE, width = 640, height = 360, print = FALSE)

Arguments

path
A character vector url/tag copied to the clipboard. Default is to read from the clipboard.
copy2clip
logical. If TRUE attempts to copy the output to the clipboard.
width
The width of the player.
height
The height of the player.
print
logical. If TRUE cat prints the output to the console. If FALSE returns to the console.

Value

Returns a character vector of an HTML iframe tag that embeds a YouTube or Vimeo video.

Description

Wrap a YouTube/Vimeo tag or url to generate an HTML iframe tag.

Details

Use YT for YouTube videos and VM for Vimeo videos.

Examples

YT("AZnVM0_ru7o", print = TRUE)







YT("http://www.youtube.com/watch?v=AZnVM0_ru7o", print = TRUE)







VM("http://vimeo.com/54007714", print = TRUE)
