Slidify Shortcut to Include reveal.js Animated Fragments

Usage

RF(print = FALSE, copy2clip = TRUE)

Arguments

print
logical. If TRUE cat prints the output to the console. If FALSE returns to the console.
copy2clip
logical. If TRUE attempts to copy the output to the clipboard.

Description

Slidify uses to include animated fragments (see this example) in reveal.js slides. This is required per each slide. Using `r RF()` reducaes the typing invloved with this action.
