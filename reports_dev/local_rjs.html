View reveal.js Slide Notes Locally

Usage

local_rjs(reveal.html = NULL, revealjs.loc = getOption("revealjs.loc"))

Arguments

reveal.html
The location of the reveal.js created html document. Default NULL will use the first available html file in the PRESENTATION directory.
revealjs.loc

Description

Reconfigures the HTML output of a slidify reveal.js presentation for use with a reveal.js full install. This allows the user to view slide notes without an Internet connection (i.e., local host is used as the server).

Details

slidify generated reveal.js presentations are not configured for use with a full install revealjs. This means that slide notes are not available for local use. local_rjs reconfigures the html output and optionally places the file in the user's reveal.js full install directory. Additionally the style.css from reports is copied to the css directory in the reveal.js full install and named reports.css.

For maximum efficiency configure the options for reveraljs.loc in their .Rprofile.
