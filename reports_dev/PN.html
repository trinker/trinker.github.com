Insert Presenter Notes

Usage

PN(text = "clipboard", copy2clip = TRUE, print = FALSE)

Arguments

text
A character vector or text copied to the clipboard. Default is to read from the clipboard.
copy2clip
logical. If TRUE attempts to copy the output to the clipboard.
print
logical. If TRUE cat prints the output to the console. If FALSE returns to the console.

Value

Returns a character vector wrapped with aHTML aside tags.

Description

Creates an aside of the class "notes" for reveal.js slides

Examples

PN("some fancy notes", print = TRUE)




some fancy notes 






PN("1) some\n2) fancy\n3) notes")



[1] "
\n1) some 
\n2) fancy 
\n3) notes 
\n
\n"



PN("1) some
2) fancy
3) notes", print = TRUE)
