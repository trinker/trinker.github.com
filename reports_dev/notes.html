View Notes

Usage

notes(col.width = 70, notes.col = FALSE)

Arguments

col.width
An integer value of the maximum width of columns.
notes.col
logical. If TRUE the notes column will be displayed.

Value

Returns a truncated view of user notes.

Description

Allows a truncated view of your notes (main_dir/ARTICLES/notes).
