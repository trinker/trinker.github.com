Wrap Text With HTML Font Tag

Usage

FT(..., text = "clipboard", copy2clip = TRUE)

Arguments

...
1 to 3 arguments of c(`face`, `size` and/or `color`). face accepts one of the following c("arial", "arial_black", "comic_sans_ms", "courier", "courier_new", "georgia", "helvetica", "impact", "palatino", "times_new_roman}", "trebuchet_ms", "verdanaBrowse"). size is any valid whole number. color can be any R color or hex value. \itemtextA character vector or text copied to the clipboard. Default is to read from the clipboard. \itemcopy2cliplogical. If TRUE attempts to copy the output to the clipboard.
