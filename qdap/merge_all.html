Merge Multiple Data Sets

Usage

merge_all(frames, by, na.replace = NA)

Arguments

frames
Multiple dataframes to merge together.
by
Specifications of the common column(s).
na.replace
Value to replace missing values with.

Value

Returns a dataframe with multiple dataframes merged together.

Description

Merge multiple data sets together.
