Separate a Column Pasted by paste2

Usage

colSplit(column, col.sep = ".", name.sep = "&")

Arguments

column
The pasted vector.
col.sep
The column separator used in paste2.
name.sep
Name separator used in the column (internal use within colsplit2df).

Value

Returns a dataframe of split columns.

Description

Separates a paste2 column into separate columns.

Examples

## Not run: # (foo <- paste2(CO2[, 1:3]))# colSplit(foo)# (bar <- paste2(mtcars[, 1:3], sep="|"))# colSplit(bar, col.sep = "|")# ## End(Not run)
