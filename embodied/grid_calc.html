Calculate Grid Lines

Usage

grid_calc(columns = 20, rows = columns)

Arguments

columns
The number of columns.
rows
The number of rows.

Value

Returns a list of 4: coordsThe coordinates to utilize for plotting the grid columnsThe number of columns rowsThe number of rows pointspoints and labels

Description

Calculate gridlines to plot ontop of an image.
