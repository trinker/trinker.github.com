Extract Causality Discourse Markers in Context

Usage

dm_causality(text.var, grouping.var = NULL, n.before = 1, tot = FALSE, n.after = n.before, ord.inds = TRUE, markup = c("<<", ">>"), name = "causality", ...)


dm_causality_sub(text.var, grouping.var = NULL, n.before = 1, tot = FALSE, n.after = n.before, ord.inds = TRUE, markup = c("<<", ">>"), name = "causality_sub", ...)

Arguments

text.var
The text variable.
grouping.var
The grouping variables. Also takes a single grouping variable or a list of 1 or more grouping variables.
n.before
The number of rows before the indexed occurrence.
tot
logical. If TRUE condenses sub-units (e.g., sentences) into turns of talk for that grouping.var.
n.after
The number of rows after the indexed occurrence.
ord.inds
logical. If TRUE inds is ordered least to greatest.
markup
A character vector of length two indicating the left (element 1) and right (element 2) boundary markers to use to highlight the causality discourse markers. Use c("", "") to not mark the causality discourse markers.
name
A string indicating the name to search for within the internal data sets, typically the function's name. Generally, for internal use.
inds
A list of integer indices to print context for.
...
Other arguments passed to termco.

Value

dm_causality - Returns returns a list of 2: countsA termco object of causality discourse marker counts. causalityA trans_context object of causality discourse markers in context.

dm_causality_sub - Returns returns a list of 2: countsA termco object of positive and negative causality discourse marker counts. continuationA trans_context object of continuation causality discourse markers in context. elaborationA trans_context object of elaboration causality discourse markers in context.

Description

dm_causality - Extract causality discourse markers in context.

dm_causality - Extract causality discourse markers in context typified as continuation or elaboration.

References

Halliday, M. A. K. & Hasan, R. (1976). Cohesion in English. English Language Series. London: Longman.

Alemany, L. A. (2005). Representing discourse for automatic text summarization via shallow NLP techniques (Unpublished doctoral dissertation). Universitat de Barcelona, Barcelona.

Schiffrin, D. (1987). Discourse markers. London: Cambridge University Press.

http://russell.famaf.unc.edu.ar/~laura/shallowdisc4summ/discmar
