Look at a package's vignette

Usage

p_vignette(package = "all")

p_vign(package = "all")

Arguments

package
name of package or "all" (if all returns all packages from the user's library that have vignettes)

Description

Use default "all" to see which packages have vignettes and supply a package name to view that vignette.

Examples

p_vignette()
p_vign()
## Not run: p_vign(fortunes)
