Display library packages

Usage

p_library(open = FALSE)

p_lib(open = FALSE)

Arguments

open
logical. If TRUE opens the directory of the add on packages library.

Description

Generates a vector of all available add on packages available to the user and optionally opens the user's library (this isn't necessarily where all of the available packages are stored).
