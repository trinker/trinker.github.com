Install zip/tar

Usage

p_zip(package = NULL, path = getOption("download_path"))

p_tar(package = NULL, path = getOption("download_path"))

Arguments

package
A quoted character string of the name/partial name of the package to install as zip/tar.
path
The path to the directory that contains the package. It is convenient to set download_path in .Rprofile options to the downloads directory.

Description

Installs zip/tar packages. This is convenient to set download_path in .Rprofile options to the downloads directory.
